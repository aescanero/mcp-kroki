Quick Start

# Add the Helm repository
helm repo add mcp-kroki https://aescanero.github.io/mcp-kroki
helm repo update

# Install the chart
helm install my-mcp-kroki mcp-kroki/mcp-kroki

# Install with custom values
helm install my-mcp-kroki mcp-kroki/mcp-kroki \
  --set service.type=LoadBalancer \
  --set ingress.enabled=true

Features

  • 27+ Diagram Types: PlantUML, Mermaid, GraphViz, D2, BPMN, and more
  • Auto-deployment: Optionally deploys Kroki server alongside MCP
  • Production-ready: Includes health checks, autoscaling, and resource management
  • Flexible configuration: ConfigMap-based with Ingress support
  • Multi-platform: Supports linux/amd64 and linux/arm64

Available Charts

mcp-kroki

Main chart for deploying MCP Kroki server

helm install my-mcp-kroki mcp-kroki/mcp-kroki --version 1.0.0

Configuration

The following table lists key configurable parameters:

  • replicaCount - Number of replicas (default: 1)
  • image.tag - Docker image tag (default: chart appVersion)
  • service.type - Kubernetes service type (default: ClusterIP)
  • ingress.enabled - Enable Ingress (default: false)
  • kroki.enabled - Deploy Kroki server (default: true)
  • config.krokiUrl - External Kroki URL if kroki.enabled=false

Examples

Deploy with External Kroki

helm install my-mcp-kroki mcp-kroki/mcp-kroki \
  --set kroki.enabled=false \
  --set config.krokiUrl=https://kroki.example.com

Deploy with Ingress

helm install my-mcp-kroki mcp-kroki/mcp-kroki \
  --set ingress.enabled=true \
  --set ingress.className=nginx \
  --set ingress.hosts[0].host=mcp-kroki.example.com

Supported Diagram Types

  • Block Diagram Family: blockdiag, seqdiag, actdiag, nwdiag, packetdiag, rackdiag
  • General Purpose: bpmn, bytefield, c4plantuml, d2, dbml, ditaa, erd, excalidraw
  • Code & Architecture: graphviz, mermaid, nomnoml, pikchr, plantuml, structurizr
  • Specialized: svgbob, symbolator, umlet, vega, vegalite, wavedrom, wireviz
